Tutorial: Balanço Financeiro Mensal (manual)

Guia Completo para Gerenciar suas Finanças com Eficiência

1. Visão Geral da Página

A página foi desenvolvida para auxiliar no controle financeiro mensal, permitindo o registro de receitas e despesas, o cálculo automático dos totais e a geração de um PDF com o balancete final. Ela conta com seções para definir o período (mês e ano), inserir o Saldo Inicial, registrar transações e visualizar os totais atualizados, além de modais informativos para esclarecimentos sobre cada item.

2. Seleção do Período e Atualização do Título

Passo 2.1: Seleção do Mês

  • Local: Na parte superior da página, dentro do cabeçalho principal (balancete).
  • Como fazer: Selecione o mês desejado no campo “Mês”. Note que o mês atualmente selecionado é “Novembro”, mas você pode escolher qualquer um na lista.

Passo 2.2: Definir o Ano

  • Local: Ao lado da seleção do mês.
  • Como fazer: Informe o ano desejado no campo numérico (por padrão, vem preenchido com “2024”).

Passo 2.3: Atualizar o Título

  • Botão: Clique no botão “Atualizar”.
  • O que acontece: O título da página (na área “Balancete Mensal”) será atualizado para refletir o mês e ano escolhidos, por exemplo: “Balancete Mensal - Novembro 2024”.

3. Inserção e Formatação do Saldo Inicial

Passo 3.1: Inserir o Saldo Inicial

  • Local: Logo abaixo do título do balancete.
  • Como fazer: No campo “Saldo Inicial (banco)”, insira o valor disponível no início do mês. Exemplo: Digite “R$ 500,00”.

Passo 3.2: Formatação Automática

  • Quando: Ao sair do campo (evento “blur”) ou pressionando a tecla Enter.
  • O que acontece: O valor inserido é formatado automaticamente para o padrão de moeda brasileiro (R$). Se o valor inserido não for válido, o campo será reajustado para “R$ 0,00”.

4. Registro de Transações (Receitas e Despesas)

4.1 Seção de Receitas

  • Estrutura da Tabela:
    • DATA: Insira a data da receita (tipo data).
    • DESCRIÇÃO: Informe uma breve descrição da entrada financeira.
    • VALOR: Digite o valor da receita (campo com formatação de moeda, que será atualizado automaticamente).
  • Adicionar Linhas: Clique no botão “Adicionar Linha” para incluir uma nova linha na tabela de receitas.
  • Remover Linhas: Clique no botão “Remover Linha” para excluir a última linha inserida.
    Observação: A remoção não pode eliminar todas as linhas; pelo menos uma linha deve permanecer.
  • Navegação entre os Campos: Pressione a tecla Enter para navegar entre os campos. Se o usuário estiver no último campo e pressionar Enter, uma nova linha será adicionada e o foco passará para o novo campo.

4.2 Seção de Despesas

  • Estrutura da Tabela:
    • DATA: Insira a data da despesa.
    • DESCRIÇÃO: Informe o tipo ou motivo da despesa.
    • VALOR: Digite o valor da despesa com formatação automática.
  • Adicionar/Remover Linhas: Mesmo procedimento da seção de receitas.
  • Formatação e Navegação: Igual ao processo explicado na seção de receitas.

5. Cálculo e Visualização dos Totais

Passo 5.1: Cálculo Automático

  • Como funciona: A cada modificação nos valores (Saldo Inicial, receitas ou despesas), a função calculateTotals é chamada para recalcular:
    • Saldo Inicial: Valor digitado e formatado.
    • Total de Receitas: Soma de todos os valores inseridos na tabela de receitas.
    • Total de Despesas: Soma de todos os valores inseridos na tabela de despesas.
    • Saldo Atual: Calculado como Saldo Inicial + Total de Receitas - Total de Despesas.

Passo 5.2: Visualização

  • Local: Os totais são exibidos em uma tabela de resumo na seção “Totais”.
  • Formato: Cada valor é exibido no padrão monetário brasileiro.

7. Exportação dos Dados para PDF

Passo 7.1: Preparação para Exportação

  • Botão: Clique no botão “Exportar Dados” na seção de totais.
  • O que acontece: Todo o conteúdo (transações e totais) é clonado e processado:
    • As datas são formatadas para o padrão DD/MM/YYYY.
    • Elementos com a classe “no-print” são ocultados no PDF.
    • Algumas classes são aplicadas para evitar quebras de página indesejadas.

Passo 7.2: Geração do PDF

  • Processo: Utilizando a biblioteca html2pdf.js, o conteúdo é convertido e salvo como “balanco-mensal.pdf”.
  • Resultado: Um arquivo PDF pronto para download ou impressão.

8. Considerações Finais

  • Interatividade: A página permite a inserção dinâmica de dados, com navegação facilitada e atualização automática dos totais.
  • Responsividade: Adapta-se bem a diferentes tamanhos de tela.
  • Exportação: A funcionalidade de exportação em PDF facilita o compartilhamento ou impressão do balancete final.
